Générateur de description de poste SFIA

Instructions

Ceci est l'outil de génération de description de poste, basé sur le Cadre SFIA 8 Framework.

Pour commencer à générer une description de poste, sélectionnez les niveaux des ensembles de compétences dont vous avez besoin.

Vous pouvez afficher une description de chaque compétence en survolant celle-ci, ou une description de chaque niveau de compétence disponible en survolant sa case respective.

Une représentation formatée de votre sélection sera générée pour vous à la fin de la page. Copiez-collez celle-ci dans un traitement de texte pour préserver la mise en forme.

Vous avez également la possibilité d'exporter votre sélection vers un document au format de valeurs séparées par des virgules (CSV) ou en HTML, qui sera automatiquement téléchargé pour vous. Cette fonctionnalité a des limitations dans certains navigateurs, nous vous recommandons donc d'utiliser une version récente de Chrome ou Firefox si vous souhaitez utiliser cette fonctionnalité, sinon le fichier pourrait être incorrectement nommé "Unknown", bien que le contenu soit le même.

Pour enregistrer votre résumé pour une utilisation future, il vous suffit de mettre la page en favori une fois votre sélection effectuée. Le lien du favori conservera toutes les informations nécessaires et mettra à jour votre sélection si vous y revenez ultérieurement.
